카카오맵으로 반경 500m 내(교차로는 200m, 대형마트·백화점·멀티플렉스는 최대 1,000m) 실측 수집한 값입니다. 카페·저가형카페 수는 유사도 매칭에, 지하철거리·교차로 인접 여부는 유사도 매칭의 "지하철·교차로 접근성" 항목에 반영됩니다. 교차로는 "OO사거리/교차로" 명칭 검색으로 판단하며, 횡단보도 자체는 카카오맵 API로 확인이 안 되는 정보라 정확한 대체 지표는 아닙니다. ※ 버스정류장 항목은 뺐습니다 — 카카오맵 공식 카테고리에 없고, 정류장이 노선별 고유명사로 등록돼 있어 "버스정류장"이라는 키워드로는 신뢰성 있게 조회가 안 됐습니다. 대형마트·백화점·멀티플렉스·공영주차장은 "주변 집객시설" 목록과 지도 마커로 표시됩니다. ※ 카페·음식점 수는 최대 45개까지만 집계됩니다 — 카카오 지도 API가 한 카테고리당 최대 3페이지(15개×3)까지만 결과를 내주는 자체 상한이라, 매장이 45개에 도달하면 실제로는 그보다 훨씬 많더라도 "45개"로 동일하게 표시됩니다 (강남역급 초대형 상권일수록 이 상한에 걸릴 가능성이 높습니다).

유동인구 & 소비자 정보 — 소상공인365(bigdata.sbiz.or.kr)에서 수집

📊 매출 데이터 자동 반영 — 매출대시보드 html 파일 업로드 시 전월매출·3개월평균·6개월평균 자동 갱신
매달 마감된 매출대시보드(html) 파일을 그대로 선택하면, 파일 안의 매출 데이터를 읽어서 점포명이 일치하는 매장의 전월매출·3개월평균·6개월평균·영업개월수를 자동으로 갱신합니다. 새로 생긴 매장(매출대시보드에는 있지만 여기 목록에 없는 경우)은 매칭 안 됨으로 안내되니, "신규 매장 추가"로 별도 등록해주세요. 반영 결과는 이 브라우저에 저장되며, 영구 반영하려면 하단 "전체 STORES 코드 내보내기"를 사용하세요.
➕ 신규 매장 추가 — 주소 입력 시 상권데이터·상권유형 자동 분석
상권유형은 이름이 아니라 위치 기반으로 자동 분류됩니다 (매장 추가 시 지도상 실제 반경을 분석). 우선순위: ① 지하철역 300m 이내 → 역세권 ② 대학교 500m 이내 → 대학가 ③ 법원·시청 등 관공서 300m 이내 → 오피스·관공서 ④ "종합병원" 명칭 300m 이내(개인 의원·동물병원 제외) → 병원상권 ⑤ 해당없음 → 일반상권. 분류 근거(거리·시설명)는 추가 완료 후 표에서 확인할 수 있습니다.
영업개월수는 오픈일자로 자동 계산됩니다. 오픈월은 영업일이 20일 이상일 때만 1개월로 인정됩니다 (예: 1/20 오픈 시 1월은 영업일 12일뿐이라 미포함, 2월부터 카운트). 이 개월수 기준으로 3개월 이하는 전월매출만, 3개월 초과 6개월 미만은 3개월평균까지, 6개월 이상은 전체 입력이 활성화됩니다.
추가한 매장은 이 브라우저에 자동 저장됩니다(새로고침해도 유지). 다른 컴퓨터·배포본에도 영구 반영하려면 "전체 STORES 코드 내보내기"로 복사한 코드를 index.html의 const STORES = [...] 부분에 붙여넣고 GitHub에 업로드하세요.
STORES 코드 (클립보드에 복사됨)
🧑‍🤝‍🧑 매장별 유동인구 데이터 입력 — 유사도 매칭 정확도 향상용
지금까지 유사매장 매칭(findSimilar)은 지도데이터(지하철·카페밀집도 등)만으로 이루어져서, "지도상 비슷해도 실제 유동인구·소비력은 다른 매장"이 매칭되는 문제가 있었습니다. 아래에 각 매장의 유동인구·편의점매출·저가형매장매출·거주인구·여성비율·20~30대비율·피크시간대를 소상공인365에서 조회해 입력해주시면, 매칭 정확도가 올라갑니다. 입력 안 한 매장은 기존처럼 지도데이터만으로 매칭됩니다(하위호환).
값은 이 브라우저에 자동 저장됩니다. 입력 후 반드시 "입력값 저장"을 눌러주세요.

떠기다방 기존 점포 현황 (29개점)
이 표에 반영된 데이터: 층수·월세·면적은 수기 입력값이고, 상권유형·지하철거리·카페·저가형카페는 각 매장 실제 위치를 기준으로 카카오맵에서 자동 수집·분석한 값입니다. 상권유형은 지하철역(300m)〉대학교(500m)〉관공서(300m)〉종합병원(300m, 개인 의원 제외) 순으로 자동 분류됩니다 (뱃지에 마우스를 올리면 분류 근거 확인 가능). 이 항목들이 전부 "신규 입지 분석"의 유사매장 매칭·예상매출 계산에 그대로 쓰입니다 — 즉 여기 보이는 수치가 다른 탭 분석 결과의 실제 근거 데이터입니다. 카페 수는 카카오 API 자체 상한으로 최대 45개까지만 집계되니, "45개"로 표시된 곳은 실제로 더 많을 수 있습니다.

영업개월수 기준: 3개월 이하는 전월매출만 표시, 3개월 초과는 3개월평균 추가, 6개월 이상은 6개월평균까지 표시됩니다 (안정적인 표본 크기가 될 때까지 평균을 보류하는 방식). 주황색 "(추정)" 표시는 아직 실매출이 없는 신규오픈 매장으로, 상권환경이 가장 비슷한 기존 매장 5곳의 평균으로 추정한 값입니다 — 실제 매출이 아니니 참고용으로만 활용하세요.
⚖️ 상권 비교함
"신규 입지 분석"에서 주소를 분석한 뒤 "⭐ 이 입지 비교함에 저장" 버튼을 누르면 여기에 쌓입니다. 2~3곳 정도 저장해서 나란히 비교해보세요. 각 항목에서 가장 좋은 값은 초록색으로 강조 표시됩니다. 비교가 끝나면 "초기화"로 전부 지우고 새로 시작할 수 있습니다.

🗾 저가형카페 분포 지도 (화이트스페이스 탐색)
지도를 원하는 지역(시/군/구 단위 권장)으로 이동·확대한 뒤 "현재 화면 기준 분포 보기"를 누르면, 그 화면 범위 안의 저가형카페 브랜드 매장을 전부 찾아서 표시합니다. 마커가 몰린 곳은 이미 경쟁이 치열한 곳이고, 마커가 비어있는 곳은 아직 저가형카페가 진출 안 한 공백지(화이트스페이스) 후보입니다.

주의: 이건 저가형카페 유무만 보여주는 것이라, 실제로 그 자리가 좋은 입지인지는 별도로 "신규 입지 분석"에서 유동인구·경쟁·상권유형까지 확인해야 합니다. 마커가 없다고 무조건 좋은 자리는 아니고, 애초에 상권 자체가 형성 안 된 곳일 수도 있습니다.
